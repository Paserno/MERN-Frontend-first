En `router/AppRouter.js`

  BrowserRouter as Router,

* Se crea las 2 rutas hacia `LoginScreen` y `CalendarScreen`, ademas de un `Redirect` en el caso que se mande una ruta desconocida.

      </div>
    </Router>
  )
}
